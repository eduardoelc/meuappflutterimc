# Criando uma Calculadora de IMC Com Flutter

## Funcionalidades
- Formulario
    - Nome
    - Peso
    - Altura
- Calulo realizado
``````
    IMC = peso / (altura * altura)
``````
- Retorne
``````
    Nome: Eduardo
    Peso: 60.0
    Altura: 1.75
    IMC: 19.59
    Classificação: Peso normal
``````
